Privacy Policy for Easy JSON Prettier

Last updated: June 19, 2026

Easy JSON Prettier is a Safari Web Extension that detects raw JSON responses by looking at a page's actual content, and renders them as a readable, collapsible, syntax-highlighted view. This policy explains what data the extension accesses, how it is used, and why it requests the permissions it does.

Summary

Easy JSON Prettier does not collect, transmit, sell, or share any of your data. It runs entirely on your device. It has no servers, no analytics, no crash reporting, and no third-party SDKs, and it makes no network requests of its own.

What the extension does

Easy JSON Prettier examines the content actually rendered on a page — never the URL, file extension, or HTTP headers — and checks whether that content, in its entirety, is a valid JSON document. If it is, the extension replaces the plain-text display with a formatted tree view. If the page is not entirely JSON, the extension makes no changes and the page is left completely untouched.

Why the extension requests access to all websites (<all_urls>) and all frames (all_frames)

Easy JSON Prettier cannot know in advance which website, API endpoint, or embedded frame might return a raw JSON response — automatic, content-based detection only works if it can run anywhere, rather than being limited to a fixed list of sites. Specifically:

  • <all_urls> lets the extension's content script run on any page so it can check whether that page's own content is JSON, regardless of domain.
  • all_frames extends the same check to content loaded inside iframes — for example, a JSON endpoint embedded in a frame — so it is formatted consistently with the top-level page.

This access is used solely to inspect a page's own rendered text, locally, in order to decide whether to format it. The extension does not read form fields, passwords, cookies, browsing history, or any other page data, and it never transmits page content anywhere — there is no network code in the extension at all.

Data storage

The extension stores two small settings using Safari's local extension storage (chrome.storage.local), which stays on your device and is never synced to a server or transmitted anywhere:

  • Whether the extension is enabled globally
  • The list of hostnames where you've chosen to turn formatting off

No other data is stored.

Clipboard and file downloads

The "Copy" and "Save" actions in the formatted view only run when you click them. They write the currently displayed JSON to your clipboard or save it to a file you choose. The extension never reads your existing clipboard contents, and neither action involves a network request.

Third parties

Easy JSON Prettier does not use analytics, advertising, crash reporting, or any third-party services or SDKs of any kind.

Children's privacy

The extension does not knowingly collect data from anyone, including children, because it does not collect data at all.

Changes to this policy

If this policy changes, the "Last updated" date above will be revised. Continued use of the extension after a change constitutes acceptance of the revised policy.
